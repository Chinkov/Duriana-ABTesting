# ABTesting App

- API to do AB Testing.

===
### API

`POST /api/v1/variants`
```
{
  "userID": 1,
  "experiment": "signups"
}
```


`POST /api/v1/track`
```
{
  "userID": 1,
  "experiment": "signups"
}
```
